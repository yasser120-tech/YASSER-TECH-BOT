# YASSER-TECH-BOT

Simple WhatsApp bot using Meta Cloud API.

## HOW TO CONNECT

1. Go to Facebook Developers > WhatsApp Cloud API
2. Copy:
   - PHONE_NUMBER_ID
   - WHATSAPP_TOKEN
3. Replace inside index.js:
   - YOUR_PHONE_ID
   - YOUR_ACCESS_TOKEN
4. Verify webhook using:
   - /webhook
   - verify token: yasser_token
